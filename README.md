# Images pour le site G-TOPOSED

## Instructions pour ajouter les images

Ce dossier contient les fichiers d'images pour le site web G-TOPOSED. Remplacez les fichiers vides par vos images réelles.

## Images requises :

### 1. hero-bg.jpg
- **Utilisation**: Arrière-plan de la section hero
- **Dimensions recommandées**: 1920x1080px
- **Contenu suggéré**: Photo aérienne de Bangui ou paysage urbain professionnel
- **Style**: Image de haute qualité avec overlay de texte

### 2. team-photo.jpg
- **Utilisation**: Section "Qui Sommes-Nous"
- **Dimensions recommandées**: 400x400px (carré)
- **Contenu suggéré**: Photo de l'équipe G-TOPOSED avec Jean-Claude Nibayom
- **Style**: Photo professionnelle, tenue d'entreprise

### 3. lotissement-9eme.jpg
- **Utilisation**: Projet "Lotissement 9ème Arrondissement"
- **Dimensions recommandées**: 400x300px
- **Contenu suggéré**: Vue du chantier de lotissement du 9ème arrondissement
- **Style**: Photo de chantier professionnel

### 4. surveyor-work.jpg
- **Utilisation**: Projet "Gestion des Relogés"
- **Dimensions recommandées**: 400x300px
- **Contenu suggéré**: Géomètre au travail avec équipement de topographie
- **Style**: Photo d'action, équipement visible

### 5. road-construction.jpg
- ** **Utilisation**: Projet "Infrastructure Urbaine"
- **Dimensions recommandées**: 400x300px
- **Contenu suggéré**: Travaux d'infrastructure routière ou construction
- **Style**: Photo de chantier, engins de construction

## Instructions d'installation :

1. **Préparez vos images** avec les dimensions recommandées
2. **Optimisez les images** pour le web (compression sans perte de qualité)
3. **Renommez les fichiers** exactement comme indiqué ci-dessus
4. **Remplacez les fichiers vides** dans ce dossier
5. **Vérifiez l'affichage** sur le site web

## Conseils techniques :

- **Format**: Utilisez JPG pour les photos, PNG pour les logos
- **Compression**: Visez 70-80% de qualité pour un bon équilibre
- **Taille maximale**: 500KB par image pour un chargement rapide
- **Noms de fichiers**: Utilisez uniquement des minuscules et tirets

## Accessibilité :

Les alt-text sont déjà configurés dans le HTML pour :
- Décrire le contenu des images
- Améliorer le référencement SEO
- Rendre le site accessible aux malvoyants
